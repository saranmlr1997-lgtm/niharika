# Niharika Customer Download

This package lets a customer run Niharika locally on Windows.

How to open:
- double-click `launch-niharika-desktop.cmd`

What opens:
- Niharika home on `http://127.0.0.1:8080/`
- business dashboard on `http://127.0.0.1:8080/business`

Included demo login:
- Email: `sivashankarimurugesan@niharika.com`
- Password: `DemoPass123!`
- License key: `Shown after signup or available in the seeded local demo account`

Notes:
- Microsoft Edge is used in app mode when available.
- If Edge is not found, the launcher opens the dashboard in the default browser.
